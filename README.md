# Proyecto_Sakila

## Descripción del Proyecto
Este proyecto consiste en resolver un caso práctico de negocio utilizando consultas SQL en DBeaver.

## Herramientas Utilizadas
* **Base de Datos:** PostgreSQL
* **Gestor de BBDD:** DBeaver.
* **Control de versiones:** GitHub.

## Informe de Análisis
-Tras analizar la base de datos, se identificaron todas las películas con clasificación 'R' (Restringida).
-Se extrajo un listado de actores específicos por su identificador único (ID 30 al 40)
