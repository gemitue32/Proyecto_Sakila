# Proyecto_Sakila

## Descripción del Proyecto
Este proyecto consiste en resolver un caso práctico de negocio utilizando consultas SQL en DBeaver.

## Herramientas Utilizadas
* **Base de Datos:** PostgreSQL
* **Gestor de BBDD:** DBeaver.
* **Control de versiones:** GitHub.

## Informe de Análisis
Consulta 2: Tras analizar la base de datos, se identificaron todas las películas con clasificación 'R' (Restringida).
Consulta 3: Se extrajo un listado de actores específicos por su identificador único (ID 30 al 40).
Consulta 4: Se intentó realizar una comparación entre el idioma actual y el original de las películas. Sin embargo, se detectó que la columna original_language_id contiene valores nulos en su totalidad, lo que impide realizar una comparación efectiva.
